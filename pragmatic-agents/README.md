# Pragmatic Agents Workspace

This folder is the shared "brain" for all Hologram product agents.

It mirrors the Pragmatic Institute Framework:

- MARKET
- FOCUS
- BUSINESS
- PLANNING
- PROGRAMS
- ENABLEMENT
- SUPPORT

Each subfolder contains JSON/Markdown files that AI agents read and update.
The repo is synced into the Hologram runtime so multiple agents can share context.

Do NOT store code here. This is **product + market knowledge only.**
